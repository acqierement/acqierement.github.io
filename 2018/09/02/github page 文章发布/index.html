添加 SSH Key 到 GitHub

在本机设置 SSH Key 之后，需要添加到 GitHub上，以完成 SSH 链接的设置。

  • 1、打开本地 id_rsa.pub 文件（ 就是前面说的那个目录下面的id_rsa.pub文件）。此文件里面内容为刚才生成的密钥。如果看不到这个文件，你需要设置显示隐藏文件。准确的复制这个文件的内容，才能保证设置的成功。
  • 2、登陆 GitHub 系统。点击右上角的Settings—>SSH and GPG keys —> New ssh key
  • 3、把你本地生成的密钥复制到里面（ key 文本框中）， 点击 add key 就ok了
