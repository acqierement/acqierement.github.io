用github搭建这个博客过程

发表于 2017-08-12 | 更新于 2018-02-10

这篇文章主要讲述一下我搭建这个博客的过程,因为已经弄好了,这中间很多东西都是我在网上找的,很多细节也记不清了,所以只能提供大概的步骤.

创建github

包括申请账号,然后填写ssh公匙(ssh是和http类似的协议,如果用http,每次提交数据都要输入账号密码),具体的过程可以百度一下

安装git

git这个东西我现在也不太了解,它是一个版本控制器,具体的安装有各个平台的下载地址,下载安装就可以了,至于具体的git命令,稍后我会介绍要用到的git命令

仓库建立

在github界面右上角点击加号下面的new repository建立自己的仓库,仓库名一定要是usename.github.io usename就是你github的用户名,以后这个名称(usename.github.io)也是你博客的网址.其他没具体的要求,点击创建就可以了,现在你就可以有自己的仓库了

博客的搭建

将自己的仓库克隆到本地,使用git命令

git clone git@github.com:acqierement/acqierement.github.io.git

后面是仓库地址,这里写的是我的仓库地址,再强调一遍,仓库地址有http和ssh两种类型,不要复制错了.之后在你的本地就会有一个.io结尾的文件夹,点进去,将里面的文件全部删掉.但隐藏的系统文件不要删,比如有一个.git文件就要保留着.然后去下载你喜欢的模板,现在流行的有jekyll和hexo,选择一个自己喜欢的模板,下载下来,然后把模板里面的全部东西复制到原来本地的仓库地址.接下来就是把自己的代码传到github上(当然你可以先卡下一步,写一篇自己的博客再上传看看自己博客的样子)

git add –all

git commit -m “msg”(后面的msg是自己可以随便写的,用来记录你做了什么改动)

git push origin master 这个是提交的代码

git pull origin master这个是将github上更新的代码传到本地

写自己的博客

我用的是jekyll主题的,它的博客是写在_posts文件夹里面的,用markdown语法写的,具体用法请百度(请原谅我的不靠谱,因为我才刚起步),写完别忘了要用前面的语句上传到github上

当然,最后你就可以输入usename.github.io去看自己的博客了
